# Sentiment Analysis Flask Web App

## Project Description
This project predicts whether input text sentiment is positive or negative using machine learning.

## Technologies Used
- Python
- Flask
- HTML/CSS
- Scikit-learn
- NLTK

## Features Added
- Modern UI
- Threading
- Unit Testing
- Postman API Testing
- Error Handling

## Original Repository Credit
Original project based on:

## Deployment
Runs on CPU without GPU.
